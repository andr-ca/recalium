# recalium

Recalium is a local-first, MCP-enabled personal memory platform that captures user interactions across LLMs, agents, and tools, then transforms them into durable, searchable, and intelligent context for future interactions.

Its purpose is to provide a portable, intelligence-assisted memory layer that persists across models, sessions, and applications while remaining inspectable, user-controlled, and source-backed.

## Key Features
- **Zero-Friction Ingestion:** Automated capture via "Recalium Capture" browser extension and watched folders.
- **Intelligent Memory:** Automated conflict detection, temporal decay, and confidence-scored curation.
- **Source-Backed Provenance:** Every piece of memory is linked back to its original source.
- **Local-First & Private:** Defaults to local storage and supports local-only processing (Ollama/Sentence Transformers).
- **MCP Enabled:** Seamlessly expose your project history and preferences to any MCP-compatible agent or tool.
